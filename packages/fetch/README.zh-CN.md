## 🚀 特性

- **Node.js 优化** - 专为 Node.js 服务端环境设计
- **智能缓存** - 内置智能缓存机制
- **进度追踪** - 支持下载进度与 CLI 进度条
- **TypeScript 支持** - 完整的 TypeScript 类型安全
- **基于 Axios** - 基于可靠的 Axios 库构建
- **轻量级** - 极小的体积，包含核心功能

## 📦 安装

```bash
# npm
npm install @esmx/fetch

# pnpm
pnpm add @esmx/fetch

# yarn
yarn add @esmx/fetch
```
