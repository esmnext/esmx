## 🚀 特性

- **通用 Vue 支持** - 支持 Vue 2.7+ 和 Vue 3 应用
- **高性能构建** - 基于 Rspack 的超快构建，配有 Vue 优化
- **模块联邦** - 内置模块联邦能力，支持微前端架构
- **SSR 支持** - 完整的 Vue 应用服务端渲染支持
- **热模块替换** - Vue 组件热重载，快速开发体验
- **TypeScript 就绪** - 完整的 TypeScript 支持，出色的开发体验

## 📦 安装

```bash
npm install @esmx/rspack-vue -D
```
