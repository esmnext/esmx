## 🚀 特性

- **原生微前端** - 基于 ESM + ImportMap 的原生模块加载，零运行时开销
- **依赖管理** - 中心化的依赖管理，基于内容哈希的强缓存策略
- **应用隔离** - ESM 原生模块隔离，保障应用运行时的稳定性
- **SSR 支持** - 灵活的服务端渲染策略，支持任意前端框架
- **开发体验** - 完整的 TypeScript 支持，原生模块链接能力
