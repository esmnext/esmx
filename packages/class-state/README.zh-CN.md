## 🚀 特性

- **基于类的设计** - 简洁直观的基于类的状态管理
- **框架无关** - 适用于任何 JavaScript 框架或原生 JS
- **TypeScript 就绪** - 完整的 TypeScript 支持，出色的类型推断
- **响应式系统** - 内置响应性，自动更新 UI
- **轻量级** - 最小化包体积，最大化功能
- **现代 API** - 使用现代 JavaScript 特性和模式

## 📦 安装

```bash
npm install @esmx/class-state
```
