## 🚀 特性

- **模块联邦** - 为微前端架构提供高级模块联邦能力
- **动态链接** - 运行时模块链接和解析
- **性能优化** - 为生产环境优化，开销最小
- **TypeScript 支持** - 完整的 TypeScript 支持，类型安全
- **热重载** - 支持开发过程中的热模块替换
- **Rspack 集成** - 与 Rspack 构建系统原生集成

## 📦 安装

```bash
npm install @esmx/rspack-module-link-plugin -D
```
