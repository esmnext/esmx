## 🚀 特性

- **Stylelint 集成** - 为现代 Web 开发提供全面的 CSS 检查规则
- **多格式支持** - 支持 CSS、Less、Vue SFC 样式文件的优化配置
- **属性排序** - 自动 CSS 属性重排序，提高代码可维护性
- **现代标准** - 基于最新的 Web 标准和最佳实践
- **Vue.js 就绪** - 专门支持 Vue 单文件组件
- **零配置** - 开箱即用，提供合理的默认配置

## 📦 安装

```bash
npm install @esmx/lint -D
```
