# ssr-rsbuild-html

An Esmx example: a no-framework HTML application with Server-Side Rendering, built with **Rsbuild**.

## 📦 Tech Stack

- **Framework**: [Esmx](https://esmx.dev) — micro-frontend framework based on native ESM
- **Build Tool**: Rsbuild ([`@esmx/rsbuild`](https://github.com/esmnext/esmx/tree/master/packages/rsbuild))
- **UI**: None (plain TypeScript + HTML)
- **Rendering Mode**: Server-Side Rendering (SSR)

## 🚀 Quick Start

```bash
npm install
npm run dev      # development server at http://localhost:3001
npm run build    # production build
npm run start    # run the production server
```
