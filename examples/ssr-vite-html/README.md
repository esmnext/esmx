# ssr-vite-html

An Esmx example: a no-framework HTML application with Server-Side Rendering, built with **Vite**.

## 📦 Tech Stack

- **Framework**: [Esmx](https://esmx.dev) — micro-frontend framework based on native ESM
- **Build Tool**: Vite ([`@esmx/vite`](https://github.com/esmnext/esmx/tree/master/packages/vite))
- **UI**: None (plain TypeScript + HTML)
- **Rendering Mode**: Server-Side Rendering (SSR)

## 🚀 Quick Start

```bash
npm install
npm run dev      # development server at http://localhost:3000
npm run build    # production build
npm run start    # run the production server
```
