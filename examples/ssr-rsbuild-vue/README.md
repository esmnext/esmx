# ssr-rsbuild-vue

An Esmx example: a Vue 3 application built with **Rsbuild**.

## 📦 Tech Stack

- **Framework**: [Esmx](https://esmx.dev) — micro-frontend framework based on native ESM
- **Build Tool**: Rsbuild ([`@esmx/rsbuild-vue`](https://github.com/esmnext/esmx/tree/master/packages/rsbuild-vue))
- **UI Framework**: Vue 3
- **Type Checking**: TypeScript

## 🚀 Quick Start

```bash
npm install
npm run dev      # development server at http://localhost:3005
npm run build    # production build
npm run start    # run the production server
```
