# ssr-rsbuild-react

An Esmx example: a React application built with **Rsbuild**.

## 📦 Tech Stack

- **Framework**: [Esmx](https://esmx.dev) — micro-frontend framework based on native ESM
- **Build Tool**: Rsbuild ([`@esmx/rsbuild-react`](https://github.com/esmnext/esmx/tree/master/packages/rsbuild-react))
- **UI Framework**: React 18
- **Type Checking**: TypeScript

## 🚀 Quick Start

```bash
npm install
npm run dev      # development server at http://localhost:3004
npm run build    # production build
npm run start    # run the production server
```
