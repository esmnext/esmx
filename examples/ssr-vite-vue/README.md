# ssr-vite-vue

An Esmx example: a Vue 3 application built with **Vite**.

## 📦 Tech Stack

- **Framework**: [Esmx](https://esmx.dev) — micro-frontend framework based on native ESM
- **Build Tool**: Vite ([`@esmx/vite-vue`](https://github.com/esmnext/esmx/tree/master/packages/vite-vue))
- **UI Framework**: Vue 3
- **Type Checking**: TypeScript

## 🚀 Quick Start

```bash
npm install
npm run dev      # development server at http://localhost:3003
npm run build    # production build
npm run start    # run the production server
```
