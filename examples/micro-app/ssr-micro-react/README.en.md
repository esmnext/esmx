# ssr-micro-react

React sub-app built with React 18 + Hooks.

## Description

This package demonstrates how to integrate a React 18 application into a micro-frontend architecture using `@esmx/router-react`'s `RouterProvider` and `RouterView` for Esmx Router integration.

## File Structure

- `src/app.ts` - App factory function creating React app instance
- `src/routes.ts` - Route configuration
- `src/entry.node.ts` - Node.js entry configuration

## Routes

- `/react` - React micro-app page
