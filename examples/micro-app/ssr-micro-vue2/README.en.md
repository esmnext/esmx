# ssr-micro-vue2

Vue 2 sub-app built with Vue 2.7 + Composition API.

## Description

This package demonstrates how to integrate a Vue 2.7 application into a micro-frontend architecture using `@esmx/router-vue`'s `RouterPlugin` and `useProvideRouter` for Esmx Router integration.

## File Structure

- `src/app.ts` - App factory function creating Vue instance
- `src/app.vue` - Vue root component
- `src/routes.ts` - Route configuration
- `src/entry.node.ts` - Node.js entry configuration

## Routes

- `/vue2` - Vue 2 micro-app page
